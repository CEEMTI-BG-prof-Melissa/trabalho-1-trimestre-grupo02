Requisitos Funcionais ༊*·˚

  1. O sistema deve ser de fácil acesso.
  2. O sistema deve mostrar os livros disponíveis e não disponíveis.
  3. O sistema deve notificar o usuário a data da devolução dos livros.
  4. O sistema deve disponibilizar livros digitais.
  5. O sistema deve recomendar livros baseado nos que o usário ja leu.
  6. O sistema deve ter um ranking dos livros mais lidos pelos alunos.
  7. O sistema deve reservar um exemplar antecipadamente.
  8. O sistema deve ter um espaço para discussões sobre os livros.
  9. O sistema deve permitir que os professores indiquem leituras personalizadas.
  10. O sistema deve permitir que os professores acompanhem o progresso dos alunos.
  11. O sistema deve registrar quais livros cada aluno está lendo.
  12. O sistema deve permitir anexar arquivos extras aos livros digitais.
  13. O sistema deve ter um painel administrativo com visualização dos dados gerais.
  14. O sistema deve permitir reservar exemplares físicos.
  15. O sistema deve exibir estatisticas de uso.
  16. O sistema deve disponibilizar filtros para busca.

Requisitos Não Funcionais ༊*·˚

  1. O período de devolução dos livros deve ser de 25 dias .
  2. Os alunos podem pegar no máximo 3 livros.
  3. O cadastro dos usuários devem ser via e-mail institucional.
  4. A interface deve ser intuitiva e amigável.
  5. As informações devem ser apresentadas de forma clara e organizada.
  6. A plataforma deve funcionar 24h.
  7. Os logins devem ser diferenciados para alunos, professores e administradores.
  8. O tempo de resposta do sistema para ações comuns deve ser inferior a 2s.
  9. O backup de dados deve ocorrer automaticamente diariamente.
  10. A plataforma deve ser acessivel de diferentes dispositivos.
  11. A plataforma deve ser fácil de usar.

Requisitos Inverso ༊*·˚

  1. O sistema não deve permitir acesso sem autenticação.
  2. O sistema não permite que pessoas externas o acessem.
  3. O sistema não permite que alunos visualizem estatísticas administrativas.
  4. O sistema não deve exibir livros que não estão disponíveis
  5. O sistema não permite a reserva de mais de 3 livros por vez.
  6. O sistema não permite a reserva de livros ja reservados.
  7. O sistema não deve recomendar livros aleatoriamente sem considerar o interesse do usuário.
